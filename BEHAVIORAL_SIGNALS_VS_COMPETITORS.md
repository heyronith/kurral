## Part 1: How Current Social Media Algorithms Work

### 1.1 Facebook/Meta Algorithm

**What it optimizes for:**
- Engagement (likes, comments, shares)
- Time spent on platform
- Return visits
- Ad impressions

**How it works:**
```
Post → Extract features → Predict engagement → Rank by predicted engagement
Features:
- Author credibility (follower count, past engagement)
- Content type (video, image, text)
- Recency
- Social signals (likes, comments, shares)
```

**The problem:**
- Optimizes for engagement, not value
- Engagement ≠ truth (misinformation can be highly engaging)
- Engagement ≠ quality (clickbait is highly engaging)
- Engagement ≠ usefulness (outrage is highly engaging)

**Gaming vectors:**
- Clickbait titles
- Outrage-inducing content
- Misinformation (more engaging than truth)
- Sensationalism
- Emotional manipulation

**Result:** Misinformation spreads faster than truth because it's more engaging.

## Part 2: What's Wrong with Current Approaches

### 2.1 The Core Problem: Engagement ≠ Value

**Current algorithms optimize for:**
- Engagement (likes, comments, shares, watch time)
- Virality (how fast content spreads)
- Addiction (how much time users spend)

**But engagement is a terrible proxy for value because:**

1. **Misinformation is more engaging than truth**
   - False claims trigger emotional responses
   - Emotional responses drive engagement
   - Engagement is rewarded
   - Result: Misinformation spreads faster

2. **Outrage is more engaging than nuance**
   - Outrage triggers strong emotions
   - Strong emotions drive engagement
   - Engagement is rewarded
   - Result: Polarization increases

3. **Entertainment is more engaging than education**
   - Entertainment is immediately rewarding
   - Education requires effort
   - Engagement is rewarded
   - Result: Misinformation entertainment spreads faster than educational content

4. **Clickbait is more engaging than substance**
   - Clickbait triggers curiosity
   - Curiosity drives clicks
   - Clicks are rewarded
   - Result: Clickbait dominates

### 2.2 The Perverse Incentive Structure

```
Current algorithm:
Post → Predict engagement → Rank by engagement → Users see high-engagement posts
                                                    ↓
                                            High engagement = reward
                                                    ↓
                                    Creators optimize for engagement
                                                    ↓
                                    Misinformation, outrage, clickbait
                                                    ↓
                                    More misinformation spreads
```

**The problem:** The algorithm incentivizes creators to produce misinformation, outrage, and clickbait because those get the most engagement.

### 2.3 Why Current Approaches Fail

| Approach | Problem |
|----------|---------|
| Engagement-based | Misinformation is more engaging |
| Virality-based | False claims spread faster |
| Time-spent-based | Addictive misinformation keeps people watching |
| Upvote-based | Misinformation gets upvoted if it confirms bias |
| Follower-based | Influencers can spread misinformation at scale |
| Recency-based | New misinformation spreads before fact-checks |
